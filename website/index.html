Midbox TESTING

This is a version of Midbox made for testing and gathering feedback on features planned to be added in the next update.
Please keep in mind songs you make here will be lost due time. Only use this for testing features for Midbox's development.

Midbox is currently testing:
- New custom chip prompt QoL and improvements.
- 5 new FM operator waveforms; Clang, Metal, Rounded (UltraBox), Secant, and Double Sine.
- Strum speed, slide speed, "continue through pattern" checkbox, and arpeggio patterns.


Fixed from the previous testing version: - Removed the unison select from instruments that don't actually support it yet.
- Fixed randomly-generated instruments not generating due to missing transition/chord types.
